# 🛠️ Instruction Manager

Десктопное приложение для управления пошаговыми инструкциями, задачами и зависимостями.
Построено на стыке интерактивного UI фреймворка **Flet** и мощной **Django ORM** (работает локально как библиотека без запуска веб-сервера).

## 🚀 Технологический стек

* **UI:** [Flet](https://flet.dev/) (на базе Flutter)
* **База данных:** [Django ORM](https://www.djangoproject.com/) + [SQLite](https://sqlite.org/)

---

## 💻 Установка и запуск (Для разработчиков)

### Предварительные требования:
1. Python `>= 3.12`
2. Установленный пакетный менеджер **uv** (см. [официальную инструкцию](https://github.com/astral-sh/uv)):

   ```bash
   curl -LsSf [https://astral.sh/uv/install.sh](https://astral.sh/uv/install.sh) | sh
   ```
### 🚀 Быстрый старт

1. Клонирование репозитория
   ```Bash
   git clone https://github.com/ViKriP/instr.git
   cd instr
   ```

2. Настройка переменных окружения
   - Скопируйте шаблон .env.example в рабочий файл .env:

   ```Bash
   cp .env.example .env
   ```
_(Отредактируйте `.env` по необходимости, но для локальной разработки дефолтных значений будет достаточно)._

3. Установка зависимостей

   ```Bash
   uv sync
   ```

4. Настройка базы данных

   - Примените миграции Django для создания таблиц в SQLite:

   ```Bash
   uv run manage.py migrate
   ```
5. Запуск приложения (Desktop GUI)

   _Приложение зарегистрировано как системная команда в pyproject.toml._

   - Запустить интерфейс можно одной командой:

   ```Bash
   uv run instr
   ```
   _(Альтернативный способ: `uv run main.py`)_

6. Контроль коммитов

   - Для активации хуков выполните:

   ```Bash
   uv run pre-commit install
   ```
   _Теперь перед каждым коммитом код будет автоматически проверяться и форматироваться._

### 👨‍💻 Панель управления (Django Admin)

   - Создайте суперпользователя:

   ```Bash
   uv run manage.py createsuperuser
   ```
   - Запустите локальный сервер:

   ```Bash
   uv run manage.py runserver
   ```
   _Откройте в браузере: `http://127.0.0.1:8000/admin`_
